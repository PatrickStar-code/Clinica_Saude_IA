# 🏥 Clínica Saúde+ --- Website

## 📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um website institucional
para uma clínica de saúde com múltiplos profissionais e serviços.

O objetivo principal é: - Apresentar a clínica - Informar os serviços
oferecidos - Facilitar o contato e o agendamento de consultas

## 📄 Estrutura do Site

### 🏠 Home

-   Nome da clínica
-   Descrição breve
-   Destaque dos principais serviços
-   Botões de ação (Agendar consulta / Contato)

------------------------------------------------------------------------

### ℹ️ Sobre a Clínica

-   Apresentação institucional
-   Missão, visão e valores
-   Diferenciais

------------------------------------------------------------------------

### 👨‍⚕️ Profissionais

Contém no mínimo 6 profissionais com: - Nome - Foto - Especialidade -
Descrição breve

------------------------------------------------------------------------

### 🩺 Serviços / Especialidades

-   Lista de serviços oferecidos
-   Descrição de cada serviço
-   Imagens ilustrativas

------------------------------------------------------------------------

### 📞 Contato / Agendamento

-   Formulário ou botão de WhatsApp
-   Endereço fictício
-   Horário de funcionamento
